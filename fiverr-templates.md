# Fiverr Response Templates

## BUYER REQUEST RESPONSE TEMPLATES

### Template 1: Generic Professional Response
```
Hi [Buyer Name]!

I'd love to help with [specific task from their request]. I have [X years/months] experience in [relevant skill] and have completed [number] similar projects.

What I'll deliver:
• [Specific deliverable 1]
• [Specific deliverable 2]
• [Specific deliverable 3]

Timeline: [Your timeline - try to beat their deadline]
Budget: [Your price - try to match or justify if higher]

I can start immediately. Let me know if you have any questions!

Best,
[Your Name]
```

### Template 2: Data Entry/Virtual Assistant
```
Hi there!

Perfect timing - I specialize in [data entry/Excel/research/admin tasks]. I can complete your [specific task] within [timeframe].

What you'll get:
• 100% accuracy guaranteed
• Clean, organized deliverables
• Unlimited revisions if needed
• Fast turnaround

I'm available [hours/timezone] and can start right away. Ready to help you save time!
```

### Template 3: Writing/Content
```
Hello!

I can write [type of content] that [achieves their goal - drives traffic/engages readers/converts/etc.].

My approach:
• Research your topic/industry thoroughly
• Write in [tone - professional/casual/persuasive]
• SEO optimized (if relevant)
• Proofread and plagiarism-free

[Number] revisions included. I'll have the first draft ready within [timeframe].

Looking forward to working with you!
```

### Template 4: When Budget is Too Low
```
Hi [Name]!

Great project! For the scope you've described ([list what they want]), the typical turnaround and quality level would be [realistic price].

I can work within your [$X] budget by:
• [Reducing scope option 1]
• [Reducing scope option 2]

Or, if you can stretch to [$Y], I can deliver everything you need with [added benefit].

Let me know what works best for you!
```

## REVISION REQUESTS

### Within Scope Revision
```
Hi [Name]!

I'll revise:
• [Change 1]
• [Change 2]
• [Change 3]

I'll have the updated version ready within [timeframe]. Anything else you'd like adjusted?
```

### Out of Scope Revision (Polite Boundary)
```
Hi [Name]!

Thanks for your message! I'm happy to help with the additional work.

The original order included [list original scope]. The changes you're requesting ([list new requests]) are beyond that scope.

I can absolutely do this work! Options:
1. I can create a custom offer for $[X] to add [new features]
2. Or, if you'd like to adjust anything from the original scope, I'm happy to revise that at no extra charge

Which would you prefer?
```

## HANDLING DIFFICULT SITUATIONS

### Buyer is Unresponsive (Needs Info to Proceed)
```
Hi [Name]!

I'm excited to start your project! To deliver exactly what you need, I just need:
• [Information 1]
• [Information 2]
• [Information 3]

Once you send these over, I'll get started immediately. Looking forward to hearing from you!
```

### Buyer is Unhappy
```
Hi [Name],

I'm sorry the delivery didn't meet your expectations. Your satisfaction is my priority.

Can you help me understand:
• What specifically needs to be different?
• [Any other clarifying question]

I want to make this right. I'll revise it until you're 100% happy with the result.
```

### Late Delivery (Emergency)
```
Hi [Name],

I sincerely apologize for the delay. [Brief, honest reason - got sick/emergency/underestimated scope].

Current status: [What's done]
New delivery time: [Realistic time]

To make up for this, I'm [offering small extra/discount/expedited delivery at no charge].

I appreciate your patience and will deliver quality work worth the wait.
```

## HOW TO USE THESE TEMPLATES

1. **Don't copy/paste blindly** - Buyers can tell
2. **Customize** the [bracketed sections] for each buyer
3. **Add personal touch** - reference something from their request
4. **Keep it concise** - 3-4 short paragraphs max
5. **End with question or call-to-action**
