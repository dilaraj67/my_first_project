## Get Your Website Live in 5 Minutes (No Coding!)

### Step 1: Get Your Affiliate Links (Do This Later)

For now, the site works with placeholder links. You can add real affiliate links after launch.

Sign up for these programs:
- Motion: https://www.usemotion.com/affiliates
- Notion: https://www.notion.so/affiliates
- Todoist: https://todoist.com/affiliates

### Step 2: Deploy to Vercel (100% Free)

**Method A: Using Vercel CLI (Easiest)**

1. Open your terminal in this folder
2. Run these commands:

```bash
# Install Vercel CLI globally (one-time setup)
npm install -g vercel

# Deploy (just press Enter for all questions)
vercel
```

3. Done! You'll get a live URL like `https://your-site.vercel.app`

**Method B: Using Vercel Website**

1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "Add New Project"
4. Import this folder
5. Click "Deploy"
6. Done!

### Step 3: Share Your Site

Your site is now live! Share it on:
- Reddit (r/productivity)
- Twitter/X
- Facebook groups
- With friends

### Step 4: Replace Affiliate Links (When Ready)

Later, when you have affiliate links:

1. Open VS Code (or any text editor)
2. Press Ctrl+Shift+F (Cmd+Shift+F on Mac) to search all files
3. Search for: `#motion-affiliate`
4. Replace with your real Motion affiliate link
5. Repeat for other apps
6. Run `vercel` again to redeploy

That's it! Your affiliate site is live and earning-ready.
